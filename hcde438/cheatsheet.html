p5.js Snippets

I used this code to make something rotate - in this case, it was for the sunrays of a sun. I learned how to use the rotate() and translate() functions to do this, as well as the push and pop functions! 

        function sunrays() {
          cX = windowWidth/4;
          cY = windowHeight/4;
          translate(cX, cY);
          let angle = TAU/numCircles;
          for (let i = 0; i < numCircles; i++) {
            stroke('white');
            strokeWeight(6);
            if (i%2==0) {
              fill(253, 197, 0);
            } else {
              fill (255, 214, 10);
            }
            push();
            rotate(i * angle);
            translate(0, -(windowHeight/1.6));
            ellipse(0,0,100, windowHeight+sunDiameter);
            pop();
          }
        }

Installing Firebase using NPM & Vite

Here's a bunch of random installation things that I used in setting up my firebase!

Run "npm install vite --save-dev" in terminal, and then copy over these scripts: 

        "scripts": {
          "start": "vite",
          "host": "vite --host",
          "build": "vite build",
          "preview": "vite preview"
        }
